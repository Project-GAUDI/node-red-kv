# node-red-kv

## 概要
node-red-kvは、下記処理を行うNode-REDのカスタムノードです。

#### 1. kv-reader-single

&emsp;・デバイスのデータ読出

#### 2. kv-writer-single

&emsp;・デバイスへのデータ書込

#### 3. kv-writer-multiple

&emsp;・デバイスへのデータ書込（IP/Port複数版）
